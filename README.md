  <h1 align='center'>Sulwork Web</h1>

  <h3>Sobre:</h3>
  <p>O desafio consistiu em desenvolver uma aplicação para cadastrar um novo colaborador para o café da manhã, excluí-lo, editar suas informações e status.</p>

  <h3>Tecnologias Back-end:</h3>
  <ul>
    <li>Angular v20</li>
    <li>TypeScript</li>
    <li>Tailwindcss</li>
    <li>Cypress</li>
  </ul>

  <h3>Ferramentas:</h3>
  <ul>
    <li>Docker</li>
  </ul>

  <h3>Tutorial Front-end:</h3>
  <ol>
    <li>
      Execute o comando:
      <code>git clone https://github.com/NicNias/Sulwork-Web.git</code>
    </li>
    <li>
      Construa e inicie os containers usando o Docker Compose:
      <code>docker build -t sulwork-app .</code>
      ou qualquer outro nome para o build
    </li>
    <li>
      Acesse o Swagger pela URL:
      <code>http://localhost:4200/</code>
      <br />
      <strong>Observação:</strong> Se estiver executando a aplicação via <code>Docker</code>, use a porta <code>4201</code>. 
      Caso esteja rodando localmente (sem Docker), utilize a porta <code>4200</code>:
      <br />
      <code>http://localhost:4200/</code>
    </li>
  </ol>
